📦 Blender UV 매핑 심층 가이드: 완벽한 텍스처링을 위한 필수 과정

강의 영상

강의 내용 요약 및 텍스트 자료

1. UV 매핑 소개 및 중요성

UV 매핑은 3D 모델의 표면에 2D 텍스처를 입히기 위해 모델을 평면으로 펼치는 과정입니다. 모델링만큼이나 최종 결과물의 시각적 품질과 텍스처 왜곡 방지에 필수적인 작업입니다.

  • UV 좌표계: X, Y, Z 대신 U, V 좌표를 사용하여 2D 맵 상의 위치를 정의합니다.
  • 목표: 텍스처의 늘어짐(Stretching)이나 겹침(Overlapping) 없이 모델 표면 전체에 텍스처를 균일하게 입히는 것입니다.

2. UV 언랩(Unwrap)의 기본 원칙과 심(Seam) 설정

심(Seam) 설정의 기술

UV 매핑의 핵심은 '심(Seam)'이라 불리는 절단선을 모델에 지정하는 것입니다. 모델을 종이 상자처럼 펼칠 때 칼집을 내는 위치와 같습니다.

  • 설정 기준: 심은 최대한 눈에 띄지 않는 곳(관절, 옷깃, 모델 뒷면)에 설정하여 시각적인 방해를 최소화해야 합니다.
  • LSCM 알고리즘: Blender의 언랩(Unwrap) 기능은 설정된 심을 기반으로 최소한의 왜곡으로 3D 표면을 2D 평면에 펼칩니다.

투사 방법의 이해

복잡한 모델 대신 단순한 원통이나 상자 형태의 모델은 투사(Projection) 기능을 사용하여 빠르게 UV를 생성할 수 있습니다.

  • Cube Projection: 건물이나 박스형 오브젝트에 적합합니다.
  • Cylinder Projection: 팔, 다리 등 원통형 오브젝트에 유용합니다.

3. 효율적인 UV 공간 활용 및 텍셀 밀도 관리

텍셀 밀도 (Texel Density)의 통일

모델의 크기와 중요도에 따라 텍스처 해상도가 균일하도록 UV 아일랜드의 크기를 조절하는 것이 중요합니다.

  • 목표: 큰 오브젝트와 작은 오브젝트 모두 동일한 픽셀 밀도(Texel per unit)를 갖도록 조정합니다.
  • **확인 방법:** 텍셀 밀도 체크용 격자(Grid) 텍스처를 사용하여 시각적으로 왜곡과 밀도를 확인합니다.

UV 아일랜드 배열 및 최적화

UV 에디터에서 펼쳐진 UV 아일랜드(Island)들을 0~1 공간 내에 효율적으로 배치합니다.

  • **공간 낭비 최소화:** 아일랜드를 회전시키거나 이동시켜 텍스처 공간을 최대한 많이 활용해야 합니다.
  • **마진 설정:** 텍스처 베이킹 시 오류 방지를 위해 아일랜드 사이에 충분한 마진(간격)을 두어야 합니다.

4. 고급 UV 기능 및 마무리

Blender의 UV 스크립팅, 핀(Pin) 기능, 평균 아일랜드 크기 맞추기 등의 고급 기능을 활용하여 복잡한 모델의 UV 작업을 효율적으로 자동화할 수 있습니다. 완벽한 UV는 퀄리티 높은 텍스처링의 필수 전제 조건입니다.
